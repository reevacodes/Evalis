🔹 Unit 1: Introduction to Programming (Q1–25)

1. Machine language is written in:
A) English
B) Binary
C) Assembly
D) C
👉 Ans: B

2. Assembly language is also called:
A) 1GL
B) 2GL
C) 3GL
D) 4GL
👉 Ans: B

3. High-level languages are:
A) Machine dependent
B) Machine independent
C) Binary
D) Hardware dependent
👉 Ans: B

4. Structured programming emphasizes:
A) GOTO
B) Modular design
C) Machine code
D) Hardware
👉 Ans: B

5. Which is NOT a structured programming construct?
A) Sequence
B) Selection
C) Loop
D) Interrupt
👉 Ans: D

6. Compiler converts:
A) Source → Object code
B) Object → Source
C) Binary → Text
D) None
👉 Ans: A

7. Interpreter works:
A) Line by line
B) Whole program
C) Binary only
D) None
👉 Ans: A

8. Object code is:
A) Human-readable
B) Machine-readable
C) Pseudocode
D) Flowchart
👉 Ans: B

9. Source code is:
A) Binary
B) Machine code
C) High-level code
D) Object code
👉 Ans: C

10. Executable file is created after:
A) Compilation
B) Linking
C) Writing
D) Debugging
👉 Ans: B

11. Linker is used to:
A) Compile code
B) Combine object files
C) Execute code
D) Debug code
👉 Ans: B

12. Loader loads program into:
A) Disk
B) RAM
C) CPU
D) Cache
👉 Ans: B

13. Operating system is:
A) Hardware
B) System software
C) Application software
D) Compiler
👉 Ans: B

14. Which is an OS?
A) MS Word
B) Linux
C) Chrome
D) Photoshop
👉 Ans: B

15. Algorithm must be:
A) Infinite
B) Finite
C) Random
D) Complex
👉 Ans: B

16. Algorithm should have:
A) Only input
B) Only output
C) Input & output
D) None
👉 Ans: C

17. Flowchart represents:
A) Code
B) Algorithm graphically
C) Binary
D) Hardware
👉 Ans: B

18. Oval symbol denotes:
A) Process
B) Start/End
C) Decision
D) Input
👉 Ans: B

19. Rectangle represents:
A) Input
B) Process
C) Decision
D) Loop
👉 Ans: B

20. Diamond represents:
A) Start
B) Process
C) Decision
D) End
👉 Ans: C

21. Parallelogram represents:
A) Decision
B) Input/Output
C) Loop
D) Process
👉 Ans: B

22. Pseudocode is:
A) Machine code
B) Informal description
C) Binary
D) Assembly
👉 Ans: B

23. Debugging means:
A) Writing code
B) Removing errors
C) Compiling
D) Executing
👉 Ans: B

24. Which is faster?
A) Interpreter
B) Compiler
C) Same
D) None
👉 Ans: B

25. Which tool translates high-level language?
A) Compiler
B) Editor
C) OS
D) Loader
👉 Ans: A
