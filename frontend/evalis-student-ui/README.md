# Evalis Frontend

## Overview

This directory contains the React.js frontend application for Evalis – Unified College Assessment Portal with Secure Proctoring and Analytics.

The frontend provides role-based interfaces for:

* Students
* Teachers
* Administrators

## Features

* Authentication and Authorization
* Exam Management
* Question Bank Management
* AI Proctoring Interface
* Analytics Dashboard
* RAG-Based Question Generation Interface
* Student Performance Tracking

## Technology Stack

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* Axios
* React Router

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Build Production Version

```bash
npm run build
```

## Deployment

The frontend is deployed using Vercel.
